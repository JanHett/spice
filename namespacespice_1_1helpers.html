Typedef documentation

template<typename T_lhs, typename T_rhs, typename T_result>
using spice::helpers::enable_if_same_t = std::enable_if_t<std::is_same<T_lhs, T_rhs>::value, T_result>

Wraps std::enable_if_t in such a way that the template is enabled when the first two template arguments are equal as determined by std::is_same. When enabled, this type becomes equivalent to the third argument T_result.
